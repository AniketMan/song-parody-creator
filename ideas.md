# Song Parody Creator — Design Ideas

## Idea 1: Editorial Manuscript

**Design Movement**: Neo-Editorial / Manuscript Aesthetic  
**Core Principles**: Paper-like texture, typewriter heritage, redline editing metaphor, literary gravitas  
**Color Philosophy**: Warm off-white (#FAF8F5) background like aged paper, deep charcoal text, red accent for "edits" (like a teacher's red pen), muted gold for highlights. The emotional intent is creative authority — you're a songwriter at a desk.  
**Layout Paradigm**: Two-column manuscript layout with a thin vertical divider (like a book spine). Left page = original, right page = your parody. Minimal chrome, maximum content.  
**Signature Elements**: Red underline on changed words (redline metaphor), subtle paper grain texture, monospace font for lyrics  
**Interaction Philosophy**: Click-to-edit inline. Hover highlights the linked word pair across both columns.  
**Animation**: Subtle fade-in on paste, gentle pulse on word change, smooth scroll sync between columns  
**Typography System**: JetBrains Mono for lyrics (monospace = alignment), Space Grotesk for UI labels and headers

## Idea 2: Studio Console

**Design Movement**: Dark Mode Production Console / DAW-inspired  
**Core Principles**: Dark interface like a recording studio, functional density, channel-strip metaphor, professional tool feel  
**Color Philosophy**: Near-black background (#0F0F12), soft white text, electric cyan (#00E5FF) for active links, amber (#FFB300) for modified words. The intent is "late night in the studio" — focused, immersive, no distractions.  
**Layout Paradigm**: Full-width split panel with draggable divider. Header strip with project controls. Feels like two tracks in a DAW — original track vs. parody track.  
**Signature Elements**: Glowing cyan connection lines between linked words on hover, amber highlight badges on modified words, subtle scanline texture  
**Interaction Philosophy**: Direct manipulation — click any word to edit in place. Drag divider to resize panels. Keyboard shortcuts for power users.  
**Animation**: Neon glow pulse on hover links, smooth panel resize, typewriter cursor blink on active edit  
**Typography System**: IBM Plex Mono for lyrics, Inter for UI chrome, bold weight hierarchy for section headers

## Idea 3: Brutalist Workbench

**Design Movement**: Brutalist Utility / Industrial Workshop  
**Core Principles**: Raw function over decoration, exposed structure, high contrast, tool-first mentality  
**Color Philosophy**: Pure white background, black text, single accent — electric lime (#BFFF00) for changed/linked words. No gradients, no shadows. The intent is "no bullshit tool" — clarity above all.  
**Layout Paradigm**: Hard 50/50 split with thick black vertical border. No rounded corners anywhere. Dense, utilitarian header bar.  
**Signature Elements**: Thick underline on modified words in lime, block cursor for editing, exposed line numbers  
**Interaction Philosophy**: Everything is a text field. No modals, no dropdowns. Paste and type. Line numbers keep you oriented.  
**Animation**: Zero decorative animation. Only functional transitions — instant state changes, no easing.  
**Typography System**: Space Mono for everything (monospace only), bold for headers, regular for body. One font, total consistency.
